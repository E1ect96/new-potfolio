Моё обучение

HTML и CSS. Профессиональная вёрстка сайтов

Первый проект был посвящён основам HTML и CSS. Надо было сверстать пару статичных страниц по макету. Во время выполнения проекта я освоил такие инструменты как GitHub Desktop и Figma, а так же научился создавать семантическую и доступную вёрстку, подключать шрифты, экспортировать графику из макета и строить сетки на Flex и Grid.

HTML и CSS. Адаптивная вёрстка и автоматизация

Основно отличие второго проекта - адаптивная вёрстка. Используя принцип MobileFirst я сверстал все страницы сайта для трёх размеров экранов - мобильный телефон, планшет и компьютер используя медиазапросы. Помимо этого в проекте присутствует адаптивная и ретинизорванная графика. Здесь я впервые познакомился с методологией БЭМ, препроцессором Sass и сборщиком Gulp.

JavaScript. Профессиональная разработка веб-интерфейсов

Впервые с JavaScript я столкнулся ещё при работе с первым проектом выполняя дополнительные задания, однако по настоящему познакомился с JavaScript именно здесь. Этот проект был посвящён изучению основ JavaScript. Работая над проектом я изучил такие понятия как объекты и массивы, встроенные объекты и функции, манипуляции с DOM и обработка событий, а так же асинхронность и работа с сетью.

JavaScript. Архитектура клиентских приложений

Это проект над которым я работаю прямо сейчас. Здесь мне предстоит освоить такие понятия как SPA, MVP, структуры данных, ООП, а так же REST и REST API

React. Разработка сложных клиентских приложений

Последний проект будет посвящён изучению React - одному из самых популярных фреймворков для JavaScript.
